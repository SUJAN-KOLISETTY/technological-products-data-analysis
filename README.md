# Data Cleaning and Visualization of Technological Products Dataset

## Project Overview

This project focuses on cleaning, preprocessing, analyzing, and visualizing a technological products dataset using Python.

The objective was to perform Exploratory Data Analysis (EDA), identify patterns in the dataset, and generate meaningful insights through data visualization.

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Dataset Features

* Brand
* Device
* Model
* Country of Origin
* Date of Release
* Price (USD)

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Missing Value Analysis
4. Duplicate Record Detection
5. Outlier Detection
6. Data Transformation
7. Exploratory Data Analysis (EDA)
8. Data Visualization
9. Insight Generation

## Key Insights

* The dataset contains a balanced representation of brands.
* The United States contributes the highest number of products.
* Apple products have the highest average pricing.
* Product prices vary significantly across brands.
* Most products were released in recent years.
